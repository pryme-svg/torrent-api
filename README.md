# torrent_api

A torrent scraper and API written in python

A working instance can be found at https://torrent-api1.herokuapp.com/

## Example

https://torrent-api1.herokuapp.com/getTorrents?site=$site&query=$query

Returns torrents for $query from $site

## Running

Install dependencies

```
pip install -r requirements.txt
```

Run the server

```
python api_server.py
```
